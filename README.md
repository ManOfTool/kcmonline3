# kcmonline3
* 以HTML5的Canvas和JavaScript組成的簡易圖片拼貼工具
* 用來組成艦隊

[](https://manoftool.github.io/kcmonline3)

## 使用方法
1. 截圖
    * 在遊戲畫面上右鍵 -> 另存圖片
1. 選擇要上傳的圖片
2. 選擇幾行(Row)、幾列(Column)，點擊 Create Canvas
3. 點擊 Draw

* Slicing 原始圖片切割
    * `Top-left X`：決定切割起點的X座標
    * `Top-left Y`：決定切割起點的Y座標
    * `Width`：決定要切割的寬度(px)
    * `Height`：決定要切割的長度(px)

* Scaling 切割後的圖片縮放
    * Width：寬度縮放(%)
    * Height：長度縮放(%)
